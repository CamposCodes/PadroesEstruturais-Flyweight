# PadroesEstruturais-Flyweight
DCC078 - ASPECTOS AVANÇADOS EM ENGENHARIA DE SOFTWARE - Flyweight Pattern

## Interface de Demonstração

Adicionei uma interface HTML de demonstração do padrão Flyweight em `index.html`.

- Abra `index.html` no navegador para explorar a simulação do Sistema Café/Bar.
- Use os botões para adicionar pedidos individuais ou em massa e observar as estatísticas.
